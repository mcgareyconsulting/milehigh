# Weld Symbol Reference (third-party — reference only)

Two welding-symbol reference documents kept for drafters/reviewers interpreting weld
callouts on MHMW drawings.

| File | What it is | Copyright |
| :--- | :--- | :--- |
| `Weld Symbol Chart 1.pdf` | AWS Welding Symbol Chart — basic weld symbols, groove symbols, supplementary symbols, and the standard location of elements within a welding symbol | **© 2007 American Welding Society** (AWS A2.4:2007) |
| `Weld Symbol Simplified.pdf` | "Advanced Welding Symbols" (Chapter 10) — a welding textbook chapter on interpreting full AWS symbols (weld-all-around, field weld, weld length, intermittent/skip welds, contours, melt-through, etc.) | Third-party textbook — **© respective publisher** |

## Usage constraints

These are **third-party copyrighted** works, kept as internal reference only.

- **Do NOT inline their text into prompts or redistribute them.** Unlike the other
  MHMW-authored files in this KB, this content is not ours to embed.
- For agentic weld-callout interpretation, rely on the general AWS-symbol knowledge a
  model already has; use these PDFs as the human-facing lookup, not as prompt context.
- If a future weld-review workflow needs machine-readable weld rules, author an
  original MHMW rule file (as with the code conventions) rather than transcribing these.
