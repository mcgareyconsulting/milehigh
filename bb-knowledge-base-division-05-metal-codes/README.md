# Division 05 Miscellaneous Metals — Code Knowledge Base

Reference library of building-code and industry-standard requirements for Division 05
miscellaneous metals (stairs, railings, guardrails, structural steel, welding, finishes)
in multi-family residential construction. Synthesizes IBC 2021, AISC 360-22 / 303-22,
AWS D1.1 / D1.3, ADA (2010), and OSHA 29 CFR 1910/1926.

Intended use: a **domain-rule library** for the BB PDF reviewer's code-compliance pass
over FC drawing sets. Open-ended review rationalizes past defects without concrete rules
to check against; the tables here (riser/tread dims, guard/handrail heights and loads,
opening limits, material specs, weld and inspection criteria) are that rule source.
